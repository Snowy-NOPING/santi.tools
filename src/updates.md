# updates

## v1.3.0
- image host tab — upload files to nest.rip, copy links, open or delete them from a grid
- add your nest.rip api key in the host tab to get started
- cleaned up unnecessary files from the repo

## v1.2.0
- history tab — every completed download is logged with filename, source url, and time
- compress tab — drag in video, audio, or images and compress them with quality presets
- notes tab — write and save notes, Apple Notes style, stored per user
- cookies.json can no longer be accidentally committed to git

## v1.1.0
- clicking a service in the supported services popup now opens its website
- service chips have a proper url map for sites like bilibili, rutube, niconico, bsky, and more
- service chips now highlight purple on hover

## v1.0.0
- google oauth login and registration
- user profile panel — view your username, edit your display name
- avatar upload (PNG/JPG, 5MB limit)
- storage usage tracking — see how much you've downloaded
- cobalt api integration for media downloading
- yt-dlp fallback for youtube
- support for auto, audio, and mute download modes
- configurable video quality, audio format, and bitrate
- file remuxing with drag and drop
- processing queue to track downloads
- settings persistence across sessions
- customizable api instance url
