# Changelog

## [1.0.0] - 2025-04-26

### Added
- initial release
- cobalt API integration for media downloading
- yt-dlp fallback for YouTube
- google OAuth login and registration
- user profile panel with display name editing
- avatar upload (PNG/JPG, 5MB limit)
- storage usage tracking per user and globally
- processing queue with download status
- settings panel (video quality, audio format, codecs, etc.)
- remix view for local file remuxing
- dark purple theme
